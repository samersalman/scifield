# Phase 8 — Publication

## Phase objective

Ship the framework as a polished, installable, documented open-source tool
that other researchers can apply to their own corpora. Deliverables: a
public GitHub repository; a PyPI release such that `pip install scifield`
works; an mkdocs-material documentation site with quickstart, tutorial
walkthrough of v1 results, API reference, and contribution guidelines; a
Streamlit or Plotly Dash demo app deployed on Hugging Face Spaces or
Render; pre-computed v1 and v2 result artifacts released via Zenodo with a
DOI for citation; a `CITATION.cff`; an MIT or Apache 2.0 license; a Docker
image for users who don't want to manage Python environments; and example
notebooks for 2–3 different fields (at least one outside surgery — radiology
or oncology — to demonstrate generality). Tagged as v1.0.0. Success
criterion: a researcher unfamiliar with the project can install the tool,
run the tutorial, and apply it to their own corpus within an afternoon,
verified via an informal usability test.
